# AE2 Creative Cells

[![Minecraft Version](https://img.shields.io/badge/minecraft-1.20.1-blue)](https://www.minecraft.net/)
[![Forge](https://img.shields.io/badge/forge-47.4.0-orange)](https://files.minecraftforge.net/)

AE2 Creative Cells provides per-item creative storage cells for Applied Energistics 2.

Each creative cell is a distinct item bound to a target item type and provides an infinite source/sink for that item inside AE2 networks.

## Quick links

- Source: (local)
- Issues: https://github.com/

## Installation

1. Download the mod JAR from the `build/libs/` folder after building.
2. Place `ae2creative-1.0.0.jar` into your `%minecraft%/mods` folder.
3. Start Minecraft with Forge for 1.20.1 and ensure `appliedenergistics2` is installed.

## Configuration

The mod reads `config/ae2creative/cells.json` at startup. Example default content:

```json
["minecraft:coal","minecraft:cobblestone","minecraft:iron_ingot","minecraft:diamond","minecraft:oak_log"]
```

Add or remove target item ids to change which creative cells are registered. A game restart is required after changing the config.

## Usage

- Creative cells appear in the creative tab `AE2 Creative Cells` and via `/give` commands.
- Each cell is an item like `ae2creative:creative_cell_coal` and behaves as an infinite storage for its linked item.

## Development & Build

Build with Gradle (Windows PowerShell):

```powershell
cd d:\addonae2
.\gradlew.bat build
```

Output JAR: `build/libs/ae2creative-1.0.0.jar`
